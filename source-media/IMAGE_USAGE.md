# Website Image Naming and Usage

Use this file as the image map for website refinement. Edit the semantic image names, then run:

```bash
npm run sync-images
npm run build
```

The sync script keeps the old brochure-extracted numeric filenames working while giving you clear semantic names to edit.

## Recommended crop ratios

- Home hero: 16:9 or 21:9, wide and clean.
- Product category cards: 4:3, centered product, minimal background clutter.
- Project case cards: 16:10.
- Why Us support image: landscape image, cropped to fit a compact card, preferably a grid/substation scene.

## Current category naming

The transformer categories used by the website are:

1. Oil-Immersed Power Transformer
2. Oil-Immersed Energy-Saving Transformer
3. Dry-Type Transformer
4. Rectifier Transformer
5. Special Transformer

Do not use `Oil-Immersed Distribution Transformer` as a main category unless the final product material explicitly confirms that naming.
